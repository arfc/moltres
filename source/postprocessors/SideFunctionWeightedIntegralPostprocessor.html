SideFunctionWeightedIntegralPostprocessor

Postprocessor for calculating the weighted integral sum/average of a variable along a boundary

Overview

Computes a function-weighted integral of a variable along a boundary.

Example Input File Syntax

Input Parameters

  • boundaryThe list of boundary IDs from the mesh where this object applies

    C++ Type:std::vector<BoundaryName>

    Controllable:No

    Description:The list of boundary IDs from the mesh where this object applies

  • variableThe name of the variable which this postprocessor integrates

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the variable which this postprocessor integrates

Required Parameters

  • weight_func1The weight function in the weighted integral, e.g. the velocity variable for calculating flow-averaged temperature outflow

    Default:1

    C++ Type:FunctionName

    Unit:(no unit assumed)

    Controllable:No

    Description:The weight function in the weighted integral, e.g. the velocity variable for calculating flow-averaged temperature outflow

Optional Parameters

  • allow_duplicate_execution_on_initialFalseIn the case where this UserObject is depended upon by an initial condition, allow it to be executed twice during the initial setup (once before the IC and again after mesh adaptivity (if applicable).

    Default:False

    C++ Type:bool

    Controllable:No

    Description:In the case where this UserObject is depended upon by an initial condition, allow it to be executed twice during the initial setup (once before the IC and again after mesh adaptivity (if applicable).

  • execute_onTIMESTEP_ENDThe list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html.

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html.

(tests/postprocessors/side_function_weighted_integral.i)

[Mesh]
  [box]
    type = GeneratedMeshGenerator
    dim = 2
    nx = 200
    ny = 10
    xmin = 0
    xmax = 2
    ymin = 0
    ymax = 1
  []
[]

[Variables]
  [u]
    order = FIRST
    family = LAGRANGE
  []
[]

[AuxVariables]
  [v]
    order = FIRST
    family = LAGRANGE
  []
[]

[Kernels]
  [diff]
    type = Diffusion
    variable = u
  []
[]

[AuxKernels]
  [v_aux]
    type = FunctionAux
    variable = v
    function = parabolic_func
  []
[]

[BCs]
  [u_top]
    type = DirichletBC
    variable = u
    boundary = 'top'
    value = 1
  []
  [u_bottom]
    type = DirichletBC
    variable = u
    boundary = 'bottom'
    value = 0
  []
[]

[Functions]
  # Parabola function
  [parabolic_func]
    type = ParsedFunction
    expression = '-1 * (x - 0) * (x - 2)'
  []
[]

[Executioner]
  type = Steady
  solve_type = 'PJFNK'
[]

[Postprocessors]
  # Test integral of u=1 across top boundary, weighted by parabola function
  # Solution = 4/3
  [weight_parabolic]
    type = SideFunctionWeightedIntegralPostprocessor
    variable = u
    boundary = 'top'
    weight_func = parabolic_func
  []

  # Test integral of parabola v across top boundary, weighted by parabola function
  # Solution = 16/15
  [both_parabolic]
    type = SideFunctionWeightedIntegralPostprocessor
    variable = v
    boundary = 'top'
    weight_func = parabolic_func
  []
[]

[Outputs]
  [out]
    type = Exodus
  []
[]
