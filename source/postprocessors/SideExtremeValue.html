SideExtremeValue

Finds either the min or max variable value of a variable over a boundary.

You can optionally provide a "proxy_variable", which will change the behavior of this postprocessor to find the point at which the proxy variable reaches the max/min value, and then return the value of the specified variable at that node.

The corresponding postprocessor that finds extreme values of variables evaluated inside elements (at quadrature points) is ElementExtremeValue
