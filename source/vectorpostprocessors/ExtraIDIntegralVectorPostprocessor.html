ExtraIDIntegralVectorPostprocessor

Integrates variables based on extra element IDs

Overview

The ExtraIDIntegralVectorPostprocessor object is a vector postprocessor to integrate input variables based on multiple extra element integer IDs. First, it finds unique combinations of extra IDs, and then it computes separate integral values over input variables for elements with these unique combinations. For reactor applications, component-wise values such as pin-by-pin power distribution can be easily tallied using this object when the mesh contains the appropriate IDs.

The ExtraIDIntegralVectorPostprocessor object needs the following parameters:

  • "variable": variables that this VectorPostprocessor operates on.

  • "id_name": list of extra IDs by which to separate integrals

Example Syntax

[VectorPostprocessors]
  [integral]
    type = ExtraIDIntegralVectorPostprocessor
    variable = 'value1'
    id_name = 'assembly_id'
  []
[]
(moose/test/tests/vectorpostprocessors/extra_id_integral/extra_id_vpp.i)
