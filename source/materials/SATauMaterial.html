SATauMaterial

This is the material class used to compute the stabilization parameter tau_viscosity for the Spalart-Allmaras turbulent viscosity equation.

Overview

This material class computes strong residuals for the Spalart-Allmaras turbulence model equation and adjusts the stabilization parameter for use in pressure-stabilized (PSPG) and streamline-upwind (SUPG) kernels. SATauMaterial includes all functionality from INSADTauMaterial for incompressible Navier-Stokes flow modeling with INSAD kernels from the MOOSE Navier-Stokes physics module.
