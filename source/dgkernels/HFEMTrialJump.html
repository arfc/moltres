HFEMTrialJump

Imposes constraints for HFEM with side-discontinuous variables.

Overview

This kernel adds a residual with inter-element jumps in a discontinuous variable "u" weighted by test functions corresponding to a side-discontinuous variable "lambda":

This is useful in diffusion equations with a hybrid finite element method. Refer to DGKernels System for the full HFEM weak form for Poisson's equation.

Input Parameters

  • interior_variableinterior variable to find jumps in

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:interior variable to find jumps in

  • variableThe name of the variable that this residual object operates on

    C++ Type:NonlinearVariableName

    Controllable:No

    Description:The name of the variable that this residual object operates on
