WallDistanceAux

Computes the minimum wall distance of each node from wall boundaries

Overview

This object computes the minimum distance of each mesh node to the nearest wall node. The wall distance calculation may be slightly inaccurate for unstructured or skewed structured meshes as illustrated on the Turbulence Modeling Resource webpage.
