FissionHeatSource

Overview

This object adds the heat source term of the steady-state temperature advection-diffusion equation with coupled neutron fluxes from a -eigenvalue calculation. The power input parameter allows the user to set a fixed thermal power output regardless of the neutron flux normalization factor. Alternatively, one may choose to normalize the neutron flux instead using the normalization and normal_factor parameters for the InversePowerMethod or NonlinearEigen executioners.
