DivFreeCoupledScalarAdvection

Overview

This object adds the advection term of the delayed neutron precursor equation. can be either constant velocity values defined using the u_def, v_def, and w_def input parameters or coupled velocity variables using the u, v, and w input parameters.

    C++ Type:NonlinearVariableName

    Controllable:No

    Description:The name of the variable that this residual object operates on
