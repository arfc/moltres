sourcekernelsHeatPrecursorSource

HeatPrecursorSource

Overview

This object adds the source term of the decay heat equation where is the fraction of total power attributable to decay heat group .

Example Input File Syntax

Input Parameters

  • decay_heat_constantsDecay Heat Constants

    C++ Type:std::vector<double>

    Controllable:No

    Description:Decay Heat Constants

  • decay_heat_fractionsDecay Heat Fractions

    C++ Type:std::vector<double>

    Controllable:No

    Description:Decay Heat Fractions

  • decay_heat_group_numberThe decay heat group this kernel acts on.

    C++ Type:unsigned int

    Controllable:No

    Description:The decay heat group this kernel acts on.

  • group_fluxesAll the variables that hold the group fluxes. These MUST be listed by decreasing energy/increasing group number.

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:All the variables that hold the group fluxes. These MUST be listed by decreasing energy/increasing group number.

  • num_groupsThe total number of energy groups

    C++ Type:unsigned int

    Controllable:No

    C++ Type:NonlinearVariableName

    Controllable:No

    Description:The name of the variable that this residual object operates on
