CoupledScalarAdvection

Overview

This object adds the advection term of the delayed neutron precursor equation. can be either constant velocity values defined using the u_def, v_def, and w_def input parameters or coupled velocity variables using the u, v, and w input parameters. Note that we applied the product rule and integration by parts to obtain the following weak form:

This form is only valid if the divergence of the velocity is zero, . Boundary conditions at the outlet must be defined using OutflowBC, CoupledOutflowBC, or CoupledScalarAdvectionNoBCBC.
