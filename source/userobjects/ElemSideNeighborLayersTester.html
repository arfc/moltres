ElemSideNeighborLayersTester

User object to calculate ghosted elements on a single processor or the union across all processors.

Overview

A user object to test whether or not ghost elements are correctly reserved. element_side_neighbor_layers is used to specify the number of layers of ghosting elements that will be preserved via either a geometric or an algebraic relationship manager.

Input Parameters

    Controllable:No

    Description:Number of layers to ghost
